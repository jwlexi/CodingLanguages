Python is a programming language that was invented in 1991. According to NorthEastern University, Python has become the number one most popular programming language.
There is over 19,000 jobs available for Python programmers with an average salary of $120,000. Python could be used for artificial intelligence, financial services, and data science.
One of the main reasons it's so popular is because it's so simple to use. The following is a sample programming snippet coded in Python:

Print("Hello World")
